# 自然语言处理解析服务

本项目是中国软件杯《智能办公机器人》赛题的自然语言处理解析服务。

## 说明

该仓库现已停止维护。由于后续移动云云空间产品的接口变化以及权限问题，本项目并不具备可持续的维护性。

请仅将本项目作为赛题的解题参考而非实际可用产品。
